# Self Version

Mostra a versão atual do Susa CLI.

## Como usar

### Versão completa (padrão)

```bash
susa self version
```

**Saída:**
```
Susa CLI v1.0.0
```

### Apenas o número da versão

```bash
susa self version --number
```

**Saída:**
```
1.0.0
```

## Opções

| Opção | O que faz |
|-------|-----------|
| `-n, --number` | Mostra apenas o número da versão |

## Para que serve?

- Verificar qual versão está instalada
- Validar se precisa atualizar
- Usar em scripts de automação
