# Plugins

Sistema de gerenciamento de plugins para estender as funcionalidades do Susa CLI.

## Sobre

O sistema de plugins permite adicionar comandos externos ao CLI através de repositórios Git. Cada plugin é descoberto automaticamente e integrado ao sistema de comandos.

## Comandos Disponíveis

O gerenciamento de plugins é feito através dos seguintes subcomandos:

### [Add](add.md)

Adiciona um novo plugin ao sistema a partir de um repositório Git.

### [List](list.md)

Lista todos os plugins instalados e suas informações.

### [Remove](remove.md)

Remove um plugin instalado do sistema.

### [Update](update.md)

Atualiza um ou todos os plugins para suas versões mais recentes.
